Reset demo state?

This is destructive and irreversible. It will:

  • Terminate all in-flight order workflows in the namespace
  • Delete closed workflow history (clears the Temporal UI list) Leave closed workflow history intact
  • Truncate the orders and idempotency_keys tables
  • Clear this console's batch submission log

Temporal Cloud is a managed, shared backend — this reset never terminates or deletes workflows there. In-flight and closed workflows are left intact in the namespace; only local business data is cleared.
